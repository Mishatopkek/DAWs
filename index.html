Список программ для создание музыки:
FL Studio - редактор-секвенсер для написання музики. Музика створюється шляхом запису і зведення (звукозапису) аудіо-, або MIDI-матеріалу.
Ableton Live - програма для створення музики в класичному студійному режимі, та реальному часі.
Logic Pro - цифрова звукова робоча станція та MIDI секвенсер для Mac OS X.
Reaper - програмне середовище для аудіо виробництва, записи і аудіо інженеринг.
